# Mapa Mental: Unidad 2 - La Representación del Conocimiento

Este mapa mental resume el proceso y los métodos para estructurar el conocimiento de un experto en un formato que una computadora pueda procesar. Los temas principales son:

-   **Ingeniería del Conocimiento**: Disciplina encargada de construir sistemas basados en conocimiento.
-   **Bases Formales**: El uso de la lógica (proposicional y de predicados) como fundamento.
-   **Formas de Representación**: Técnicas como Reglas de Producción, Redes Semánticas y Marcos (Frames).
-   **Rol del Ingeniero del Conocimiento**: El profesional que sirve de puente entre el experto y el sistema.
-   **Cuello de Botella**: El principal desafío, que es la dificultad de extraer el conocimiento tácito de un experto.
